# 状态估计与滤波 —— 机器人如何知道自己在哪

> **核心定位**：状态估计是机器人的"本体感觉中枢"。所有传感器都有噪声和误差，状态估计的任务是从一堆不完美的测量数据中**推断出最接近真实的机器人状态**（位姿、速度、IMU 偏置等）。

## 第 0 章：状态估计一句话

> **大白话**：各传感器（视觉/里程计/IMU）均含噪声；状态估计融合多源观测，给出一致的位置/速度估计。

---

## 第 1 章：卡尔曼滤波 (KF) —— 最优线性估计器

### 1.1 核心思想

卡尔曼滤波用两步循环：**预测 → 更新**。

| 步 | 做什么 | 输入 |
|----|--------|------|
| **预测 (Predict)** | 用运动模型猜"我现在大概在哪" | 上一时刻状态 + 控制输入（如速度指令） |
| **更新 (Update)** | 用传感器测量纠正刚才的猜测 | GPS/相机/编码器的观测值 |

> 关键创新：**卡尔曼增益** $K$ 自动决定"更相信运动模型还是更相信传感器"——取决于谁的协方差（不确定性）更小。

### 1.2 五个核心公式

$$
\begin{aligned}
&\text{Predict:} &\hat{x}_{k|k-1} &= F\hat{x}_{k-1} + Bu_k \\
& &P_{k|k-1} &= F P_{k-1} F^T + Q \\
&\text{Update:} &K_k &= P_{k|k-1} H^T (H P_{k|k-1} H^T + R)^{-1} \\
& &\hat{x}_k &= \hat{x}_{k|k-1} + K_k(z_k - H\hat{x}_{k|k-1}) \\
& &P_k &= (I - K_k H) P_{k|k-1}
\end{aligned}
$$

> $Q$ = 过程噪声协方差（对运动模型有多不信任），$R$ = 测量噪声协方差（对传感器有多不信任），$K$ = 卡尔曼增益（0~1，0=全信预测，1=全信测量）

## 第 2 章：非线性扩展 —— EKF 与 UKF

真实的机器人系统几乎都是**非线性**的（旋转不是加法、观测模型不是线性投影）。

| 方法 | 如何处理非线性 | 优点 | 缺点 |
|------|--------------|------|------|
| **EKF** | 对非线性函数做一阶泰勒展开（线性化），求雅可比 | 简单，计算快 | 强非线性时精度差，雅可比推导繁琐 |
| **UKF** | 在状态周围采样 Sigma 点，对每个点运行非线性函数，统计结果 | 不需要雅可比，精度更高 | 计算量比 EKF 大 |

> 人形机器人常用 **EKF**：融合 IMU + 编码器 + 腿部里程计，估计基座的 6-DOF 位姿和速度。

## 第 3 章：人形机器人特有的状态估计问题

### 3.1 浮动基座估计 (Floating Base Estimation)

轮式机器人的底盘永远贴地。人形机器人的基座（躯干）是一个**6-DOF 刚体**：$(x, y, z, roll, pitch, yaw)$。估计它的位姿需要融合：

- IMU（加速度 + 角速度）
- 关节编码器 → 运动学正解（每条腿都能给出基座位姿）
- 足端触地检测（只有踩在地上的腿才能提供有效信息）

### 3.2 触地判定 (Contact Detection)

这是人形状态估计独有的核心问题——机器人的脚只有**踩在地上**时，运动学推算才有效。

> **工程阈值**：Kuavo 4 Pro 用足端力传感器 20N 作为触地判定阈值。低于 20N = 摆动态（swing），该腿的运动学观测被屏蔽。

### 3.3 腿部里程计 (Leg Odometry)

每条踩在地上的腿都能通过正运动学推算基座位姿。两条腿同时踩地时，可以取加权平均（权重 = 足端法向力 / 总力）。

## 第 4 章：因子图与平滑 (Factor Graph & Smoothing)

对于需要**全局一致性**的应用（如 [SLAM](./slam.md)），EKF 的缺点是：一旦线性化点固定，过去的错误无法纠正。

**因子图**把所有历史位姿和观测约束建模为一个概率图，用非线性优化（如 Levenberg-Marquardt）一次性求解全部历史轨迹。

$$\arg\max_{x_{1:T}} \prod_t p(z_t \mid x_t) \prod_t p(x_t \mid x_{t-1}, u_t)$$

> GTSAM / g2o / Ceres 是实现因子图的主流求解器库。

## 关键词速查

| 术语 | 解释 |
|------|------|
| **KF** | 卡尔曼滤波：线性系统的递推最优状态估计 |
| **EKF** | 扩展卡尔曼滤波：对非线性系统做一阶线性化 |
| **UKF** | 无迹卡尔曼滤波：用 Sigma 点采样处理非线性，不需雅可比 |
| **Process Noise Q** | 描述运动模型的不确定性，Q 大 = 更信任传感器 |
| **Measurement Noise R** | 描述传感器的不确定性，R 大 = 更信任运动模型 |
| **Mahalanobis Distance** | 归一化的误差度量，用于异常值检测 |
| **Factor Graph** | 概率图模型，节点=变量，边=约束，批量求解 |
| **Contact Detection** | 触地判定：区分摆动态/支撑态，人形状态估计的前提 |
