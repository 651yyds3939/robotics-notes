# PID 控制理论 (PID Control)

> **核心定位**：PID 是自动控制领域
> 👉 进阶：[经典动力学与运动控制](./dynamics_control.md) · [阻抗控制](./impedance_control.md) · [FOC 电机控制](./motor_foc.md)
>
> PID 是自动控制领域最古老、最广泛应用的控制器，没有之一。现代人形机器人的每个关节内部都跑着至少一个 PID 控制器（通常是级联三环）。理解 PID 的每一项做了什么、怎么调参，是机器人底层控制的基本功。

---

## 第 0 章：PID 一句话

> **大白话**：机器人关节当前角度和期望角度有个偏差。P（比例）说"偏差多大，我出多大力"；I（积分）说"这点小偏差一直在，我要慢慢纠正"；D（微分）说"偏差变得太快了，赶紧刹车"。三个加起来，就是 PID。

---

## 第 1 章：三项解构

$$u(t) = K_p e(t) + K_i \int_0^t e(\tau)d\tau + K_d \frac{de(t)}{dt}$$

其中 $e(t) = x_{desired}(t) - x_{actual}(t)$ 是跟踪误差。

| 项 | 做了什么 | 调大后 | 调大后的副作用 |
|----|---------|--------|--------------|
| **P**（比例） | 误差越大，输出越强 | 响应快，稳态误差减小 | 超调增加，可能振荡 |
| **I**（积分） | 累计历史误差，消除稳态偏差 | 消除稳态误差 | 响应变慢，可能积分饱和（windup） |
| **D**（微分） | 误差变化率，预判趋势做阻尼 | 抑制超调和振荡 | 放大传感器噪声，对高频抖动敏感 |

## 第 2 章：级联控制 (Cascaded Control)

机器人关节通常不是单环 PID，而是**三环级联**：

```text
期望位置 ──→ [位置环 PID] ──→ [速度环 PID] ──→ [电流环 PID] ──→ 电机
              ↑                   ↑                   ↑
         编码器反馈           速度估计            电流传感器
```

| 环 | 频率 | 功能 |
|----|------|------|
| **电流环** | 20-100kHz | 最内环。直接控制绕组电流=力矩，FOC 的核心 |
| **速度环** | 1-5kHz | 中间环。控制关节转速 |
| **位置环** | 100-1000Hz | 最外环。控制关节角度 |

> 内环的频率必须是外环的 5-10 倍，否则内环跟不上导致不稳定。

## 第 3 章：工程调参实践

### 3.1 Ziegler-Nichols 法

```text
1. 设 K_i=K_d=0
2. 逐步增大 K_p 直到系统等幅振荡，记录此时的 K_u（临界增益）和 T_u（振荡周期）
3. 查表计算：K_p = 0.6·K_u, K_i = 2·K_p/T_u, K_d = K_p·T_u/8
```

### 3.2 常见问题

| 症状 | 诊断 | 调整 |
|------|------|------|
| 响应太慢，跟不上去 | P 太小 | ↑ $K_p$ |
| 反复超调，震荡 | P 太大或 D 太小 | ↓ $K_p$ 或 ↑ $K_d$ |
| 有恒定偏差（角度总是差一点点） | I 太小 | ↑ $K_i$ |
| 启动时剧烈过冲然后慢慢回来 | 积分饱和 (Windup) | 加 Anti-windup（积分限幅/条件积分） |
| 电机有高频啸叫 | D 太大，放大传感器噪声 | ↓ $K_d$，或加低通滤波 |

### 3.3 前馈控制 (Feedforward)

> 纯 PID 是被动响应——误差产生了才补救。前馈是主动预判——根据期望轨迹的一阶/二阶导数提前输出力矩。

$$\tau = \underbrace{K_p e + K_i \int e + K_d \dot{e}}_{\text{PID 反馈}} + \underbrace{M(q_d)\ddot{q}_d + C(q_d, \dot{q}_d) + G(q_d)}_{\text{前馈：逆动力学模型}}$$

前馈 + PID 反馈 = 机器人控制的经典范式。

## 关键词速查

| 术语 | 解释 |
|------|------|
| **Overshoot** | 超调量：响应超过目标值的最大值 |
| **Steady-state Error** | 稳态误差：稳定后和目标的恒定偏差 |
| **Integral Windup** | 积分饱和：大偏差时积分累积极大，导致长时间过冲 |
| **Feedforward** | 前馈：根据期望轨迹"预判"输出，零延迟 |
| **Cascaded Control** | 级联控制：内环快速响应，外环精确跟踪 |
