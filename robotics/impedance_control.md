# 阻抗控制与导纳控制 (Impedance & Admittance Control)

> **核心定位**：经典位置控制是"目标位置跟踪"（刚性的）。阻抗控制是"手碰到东西后有个缓冲，像弹簧一样"（柔顺的）。在人形机器人上，阻抗控制是安全交互的物理底座——没有它，机器人握手能把人手指捏碎。
>
> 👉 实战笔记：[WBC 柔顺起立与重力补偿](https://github.com/651yyds3939/kuavo-dev-notes/blob/master/kuavo_notes/18.teaching_gravity_compensation.md)

---

## 第 0 章：阻抗控制一句话

> **大白话**：把机器人关节变成"虚拟弹簧+阻尼器"。受外力时顺应位移；释放后回弹。刚度越大，关节越硬（像铁棍）；刚度越小，关节越软（像橡皮筋）。人形机器人起立时用**低刚度**防止爆冲，站立后用**高刚度**保持姿态。

---

## 第 1 章：阻抗 vs 导纳 —— 同一个东西的两面

| | 阻抗控制 (Impedance) | 导纳控制 (Admittance) |
|------|---------------------|----------------------|
| 因果关系 | 测量**位移** → 输出**力** | 测量**力** → 输出**位移** |
| 物理类比 | 弹簧：压缩后回弹 | 推重物：施力后加速 |
| 适用场景 | 轻量、低摩擦、有高带宽力矩控制 | 重载、高摩擦、只能做位置控制 |
| 实现方式 | 直接力矩控制：$\tau = K(q_d - q) + D(\dot{q}_d - \dot{q})$ | 外环力控：测量外力 → 修正期望位置 → 发位置指令给内环 |

> 人形机器人通常是**导纳控制**——因为底层关节驱动器接收的是位置指令而非力矩指令。

## 第 2 章：核心公式

### 2.1 阻抗控制的动态方程

$$M_d(\ddot{x} - \ddot{x}_d) + D_d(\dot{x} - \dot{x}_d) + K_d(x - x_d) = F_{ext}$$

- $M_d$：期望惯量（虚拟质量）
- $D_d$：期望阻尼（吸收能量，防止振荡）
- $K_d$：期望刚度（弹性回复力）
- $F_{ext}$：外部施加的力

> **调参直觉**：$K$ 决定"多硬"，$D$ 决定"振动多久停"，$M$ 决定"推起来多重"。

### 2.2 关节空间实现

$$\tau = K_p(q_d - q) + K_d(\dot{q}_d - \dot{q}) + \tau_{comp}$$

- $K_p$, $K_d$：关节刚度/阻尼对角阵
- $\tau_{comp}$：重力补偿前馈项（抵消自身重量）

## 第 3 章：Kuavo 实战应用

| 场景 | 刚度设置 | 原因 |
|------|---------|------|
| **经典 WBC 起立** | **低刚度** | 防止"旱地拔葱"爆冲，让机器人像棉花一样缓缓站起 |
| **[RL 切换瞬间](./RL.md)** | 0 → 逐步加载 | 避免经典控制和 [RL](./RL.md) 输出的**阶跃**冲击 |
| **正常站立** | **高刚度** | 维持姿态不晃动 |
| **示教/拖动示教** | **零刚度**（重力补偿模式） | 靠重力补偿抵消自重，人轻轻一推手臂就跟着走 |
| **碰撞检测** | 检测到碰撞瞬间**降刚度** | 遇异常阻力自动变软，防止硬件损坏 |

## 第 4 章：在 WBC 中的角色

[全身控制 (WBC)](./dynamics_control.md) 是阻抗/导纳控制的天然载体。WBC 的多任务层级结构中，**阻抗任务**通常作为最低优先级的"兜底"约束：

```text
优先级高 → 触地约束（脚不能穿地）、关节限位、力矩上限
优先级中 → 姿态跟踪、质心轨迹
优先级低 → 阻抗柔顺（手/足端对环境的顺应性，牺牲绝对跟踪精度换安全）
```

## 关键词速查

| 术语 | 解释 |
|------|------|
| **Stiffness $K$** | 刚度：偏离目标位置时的回复力大小 |
| **Damping $D$** | 阻尼：吸收动能，防止振荡过冲 |
| **Compliance** | 柔顺性：对环境外力做适应性让步的能力 |
| **Gravity Compensation** | 重力补偿：输出力矩恰好抵消自重，关节"失重"可被轻松拖动 |
| **Admittance** | 导纳：测力→出位移，适用于位置控制型驱动器 |
