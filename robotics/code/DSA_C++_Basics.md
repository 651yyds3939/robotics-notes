掌握以下五个板块后，即可开始刷 LeetCode。

## 一、 基础语法与控制流（最基本的工具）

这是写任何代码的前提，要求熟练到像写呼吸一样自然：

* **基本数据类型：** `int`（整数）、`double`（浮点数）、`char`（字符）、`bool`（布尔值 `true`/`false`）。
* **三大控制流：**
* `if-else` 条件判断。
* `for` 循环（包括传统的 `for(int i=0; i<n; i++)` 和现代 C++ 的区间 for 循环 `for(int num : nums)`）。
* `while` 循环（常用于双指针算法和二分查找）。


* **常见运算符：** 算术运算（`+`, `-`, `*`, `/`, `%`），逻辑运算（`&&` 与、`||` 或、`!` 非），以及位运算（`&`, `|`, `^`, `<<`, `>>`，位运算在部分 LeetCode 题目中是高效解法的关键）。

---

## 二、 函数与参数传递（核心：避免超时）

LeetCode 题目通常要求**实现给定签名的函数**。须掌握 C++ **引用传递**，否则大数组按值复制易导致超时（TLE）。

* **值传递（Pass by Value）：** `void func(int x)`。函数内部复制一份一份 `x`，修改内部的 `x` 不会影响外部。
* **引用传递（Pass by Reference）：** `void func(int& x)`。加个 `&`，内部修改会直接改变外部变量。**刷题时极其常用。**
* **常量引用传递（Const Reference）：** `void func(const vector<int>& nums)`。刷题时最常见的输入参数形式。意思是：我想高效地读取这个大数组（用引用，不复制），但我保证不修改它（用 `const`）。

---

## 三、 指针与自定义结构体（通关链表和二叉树的钥匙）

刷 LeetCode 绕不开链表（Linked List）和二叉树（Binary Tree），它们在 C++ 里全部是通过指针和结构体实现的。

* **指针的基本概念：** * 明白指针变量存储的是一个内存地址。
* 取地址符 `&` 和解引用符 `*` 的含义。
* 空指针必须用 `nullptr`（不要用老旧的 `NULL` 或 `0`）。


* **结构体（Struct）与成员访问：**
* 看懂并会定义简单的结构体。
* **核心：** 明白什么时候用圆点 `.` 访问成员，什么时候用箭头 `->` 访问成员。
* 如果是普通结构体变量 `TreeNode node;`，用 `node.val`。
* 如果是结构体指针 `TreeNode* root;`，必须用 `root->val`。

## 四、 核心 STL 容器及常用函数（面试手撕效率翻倍）

C++ STL 在 Hot 100 中高频出现。以下 5 类容器的常用 API 建议熟记：

### 1. `std::vector`（动态数组，代替传统数组）

* **怎么定义：** `vector<int> nums;` 或者二维数组 `vector<vector<int>> matrix;`
* **必须掌握的函数：** * `.push_back(val)`：在末尾尾插一个元素。
* `.pop_back()`：弹出末尾元素。
* `.size()`：获取数组长度。
* `nums[i]`：像普通数组一样用下标访问。



### 2. `std::string`（字符串）

* **必须掌握的函数：** `.length()` 或 `.size()`、`.substr(start, len)`（截取子串）、`+` 运算符（拼接字符串）。

### 3. `std::unordered_map`（哈希表/字典）

* **用途：** 解决大名鼎鼎的《两数之和》等查找、计数问题。
* **必须掌握的函数：** * `mp[key] = value`：插入或修改键值对。
* `.count(key)` 或 `.find(key)`：检查某个键是否存在。



### 4. `std::unordered_set`（哈希集合）

* **用途：** 去重，或者快速判断某个元素是否出现过。
* **必须掌握的函数：** `.insert(val)`、`.erase(val)`、`.count(val)`。

### 5. `std::queue`（队列）与 `std::stack`（栈）

* **用途：** 队列用于广度优先搜索（BFS/层序遍历），栈用于深度优先搜索（DFS）和有效括号等题目。
* **必须掌握的函数：** `.push()`, `.pop()`, `.empty()`，以及队列用 `.front()` 看队头，栈用 `.top()` 看栈顶。

## 五、 递归思维（算法层面的基本功）

递归是 C++ 语法和算法的交界点。在刷树、回溯算法时，80% 的代码都是递归。

* 不必深入汇编调用栈，但应能在纸上模拟简单递归（如斐波那契、阶乘）。
* 能够清晰分清：**边界条件（什么时候 return 终止）** 和 **递推公式（怎么调用自身）**。

## 🛠️ 避坑指南：哪些高阶 C++ 语法现在“绝对不要看”？

为了不走弯路，在刷完 Hot 100 之前，看到下面这些概念**请直接绕道**：

1. **面向对象高阶：** 虚函数（`virtual`）、多态、纯虚函数、继承、构造/析构函数的深拷贝浅拷贝。
2. **现代 C++ 高阶：** 智能指针（`shared_ptr`/`unique_ptr`）、右值引用与移动语义（`std::move`）、Lambda 表达式。
3. **高级模板：** 模板元编程、`template<typename T>` 的复杂设计。
4. **系统编程：** 多线程（`std::thread`）、互斥锁（`std::mutex`）、文件输入输出流（`fstream`）。
