GNOME 扩展可通过浏览器或本地客户端安装。

在 Ubuntu 的默认桌面环境（GNOME）中，以前确实流行用浏览器安装，但现在更推荐使用**本地客户端**，因为浏览器的那个插件经常因为权限问题导致安装失败。

若通过**浏览器**安装扩展，步骤如下：

### 第一步：安装系统连接层
浏览器无法直接控制系统，需先安装 GNOME 浏览器集成组件：
```bash
sudo apt update
sudo apt install chrome-gnome-shell
```

### 第二步：安装浏览器插件
1. 使用 Chrome 或 Firefox 访问 [GNOME Extensions 官网](https://extensions.gnome.org/)。
2. 页面顶部提示安装 **"GNOME Shell integration"** 浏览器扩展，点击安装。

### 第三步：搜索并安装具体的显示插件
在官网搜索以下两个最经典、就在 WiFi 图标旁显示的插件：

1. **Vitals** (最推荐)：显示温度、电压、风扇转速、内存等，图标非常精致。
2. **System Monitor Next**：可以显示 CPU 和内存的实时小波形图。

**操作方法：** 进到插件页面，把右上角的 **"OFF"** 开关拨到 **"ON"**，点击弹出框的“安装”即可。

---

### 💡 为什么现在不推荐浏览器安装？
很多用户反馈在最新版本的 Ubuntu (22.04/24.04) 中，因为浏览器大多改成了 **Snap** 版本（一种沙盒格式），它们被限制了访问系统文件的权限，导致浏览器里的开关点不动。

**如果浏览器安装没反应，最稳妥的办法是：**
1. 在终端输入：`sudo apt install gnome-shell-extension-manager`
2. 在系统应用中打开 **“扩展管理器” (Extension Manager)**。
3. 点击 **“浏览”**，搜索 **Vitals** 直接安装。

这个本地软件的效果和浏览器官网是一模一样的，但成功率是 100%。

安装完成后，可在浏览器右上角查看 CPU 温度与占用率。
