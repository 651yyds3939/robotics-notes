# Ubuntu 目录切换与管理命令笔记

## 1. `cd` 命令简介
`cd`（Change Directory）用于在终端中切换当前工作目录。

### 基本语法
```bash
cd [目录路径]
````

### 注意

* 不带参数执行 `cd`，默认切换到用户主目录。
* 使用 `pwd` 命令可以查看当前工作目录。

---

## 2. 常用切换方法

| 命令             | 功能                                |
| -------------- | --------------------------------- |
| `cd`           | 切换到当前用户的主目录（如 `/home/用户名`）        |
| `cd /`         | 切换到根目录                            |
| `cd ..`        | 切换到上一级目录                          |
| `cd ../..`     | 上两级目录，以此类推                        |
| `cd -`         | 切换到上一次所在的目录                       |
| `cd ~`         | 切换到用户主目录，等同于 `cd`                 |
| `cd ~/目录名`     | 切换到当前用户主目录下的指定目录，例如 `~/Documents` |
| `cd ~/桌面`      | 切换到当前用户桌面目录                       |
| `cd /绝对路径/目录名` | 使用绝对路径切换                          |
| `cd 相对路径/目录名`  | 使用相对路径切换                          |

---

## 3. 结合其他命令查看目录内容

| 命令       | 功能                |
| -------- | ----------------- |
| `ls`     | 列出当前目录下文件和文件夹     |
| `ls -l`  | 列出详细信息            |
| `ls -a`  | 列出隐藏文件（以`.`开头的文件） |
| `ls -lh` | 以人类可读形式显示大小       |

**示例**

```bash
cd /usr/bin
ls -l
```

---

## 4. 快速路径技巧

1. **Tab 补全**

```bash
cd Doc<Tab>
```

自动补全匹配的目录名。

2. **使用相对路径**

```bash
cd ../../Documents
```

从当前目录向上两级再进入 Documents。

3. **使用环境变量**

```bash
cd $HOME/Documents
```

`$HOME` 表示当前用户主目录。

---

## 5. 结合 `pushd` 和 `popd` 管理目录栈

* `pushd <目录>`：进入目录并将当前目录入栈
* `popd`：返回上一个目录（出栈）

**示例**

```bash
pushd /tmp
# 当前目录切换到 /tmp
popd
# 返回之前的目录
```

> 目录栈比 `cd -` 更灵活，可以管理多级跳转。

---

## 6. 小技巧与注意事项

1. 路径中有空格时，用引号或反斜杠：

```bash
cd "My Documents"
cd My\ Documents
```

2. 用 `pwd` 查看当前路径：

```bash
pwd
```

3. `cd` 命令不会创建目录，如果目录不存在会报错。

4. 使用 `~/目录名` 可以快速进入用户主目录下的子目录：

```bash
cd ~/Documents
cd ~/Downloads
```

---

## 7. 参考命令总结

| 命令         | 功能            |
| ---------- | ------------- |
| `cd`       | 切换到用户主目录      |
| `cd /`     | 切换到根目录        |
| `cd ..`    | 返回上一级         |
| `cd -`     | 返回上一次目录       |
| `cd ~/目录名` | 进入用户主目录下的指定目录 |
| `pwd`      | 显示当前目录        |
| `ls`       | 列出当前目录内容      |
| `pushd`    | 进入目录并入栈       |
| `popd`     | 返回上一个目录       |
