## 使用建议

1. **月付试水**：先月付验证晚高峰和 AI/流媒体解锁，再考虑年付。
2. **至少备一家**：主力维护时仍有备用订阅可切换。
3. **客户端**：旧版 Clash for Windows 已停更，Linux 推荐 Clash Verge Rev / Mihomo Party / FlClash。
4. **订阅安全**：链接泄露后及时在机场后台重置订阅。
5. **域名易变**：部分机场会换域名，打不开时查官方 Telegram 公告频道。
